Release Notes

VisualVM 2.0.3 has been released on July 16th, 2020. This document describes changes introduced by this release.

System Requirements

Supported OS:

  • Microsoft Windows
  • Linux: Intel platform, ARM HFLT, AArch64
  • macOS

Required Software:

  • Oracle JDK 8~14
  • OpenJDK 8~14
  • GraalVM 19~20

Changes

Features and Enhancements:

  • Improvements in Polyglot Heap Viewer for Ruby and R languages
  • Built on Apache NetBeans Platform 11.3 (https://netbeans.apache.org)

Fixed Bugs:

  • GH-85: Java 10 - VisualVM 1.4.1 - Sampler -> CPU -> CPU Settings, doesn't filter.
  • GH-196: NPE in Ruby heap viewer
  • GH-197: HeapViewer Doesn't Notify on Loading Failure
  • GH-199: VisualVM does not detect truffle processes (does not display Graal Sampler tab)
  • GH-205: Make an option to use regular expression in Class Filter
  • GH-210: IllegalArgumentException: Option 'js.nashorn-compat' is experimental and must be enabled with allowExperimentalOptions()

Known Problems

To view all known problems please see the VisualVM Issues page.

To resolve the most common problems please follow the Troubleshooting Guide.
