Release Notes

VisualVM 2.0.4 has been released on August 18th, 2020. This document describes changes introduced by this release.

System Requirements

Supported OS:

  • Microsoft Windows
  • Linux: Intel platform, ARM HFLT, AArch64
  • macOS

Required Software:

  • Oracle JDK 8~14
  • OpenJDK 8~14
  • GraalVM 19~20

Changes

Features and Enhancements:

  • New VisualVM-GoToSource plugin to open source code from profiler/heap views in external editor/IDE

Fixed Bugs:

  • GH-213: Allow To Save Modified OQL/R Script When Closing The Heap Viewer
  • GH-215: Missing local variables in thread dump
  • GH-216: Failed to initialize view org.graalvm.visualvm.profiling.snapshot.diff.SnapshotDiffView

Known Problems

To view all known problems please see the VisualVM Issues page.

To resolve the most common problems please follow the Troubleshooting Guide.
